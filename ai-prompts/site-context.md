# Site Context & Guidelines for Valley Daycare Sites

## Project Overview
Valley Daycare Sites is a specialized web development service focused on creating professional websites for daycare centers and childcare providers in the Valley area. Our mission is to help local childcare businesses connect with families through modern, user-friendly websites.

## Target Audience
- **Primary**: Daycare center owners and administrators
- **Secondary**: Parents searching for childcare services
- **Geographic Focus**: Local Valley area and surrounding communities

## Brand Identity

### Brand Values
- Family-first approach
- Professional excellence
- Community focus
- Trustworthiness
- Warmth and approachability

### Tone of Voice
- Friendly and welcoming
- Professional but not corporate
- Supportive and helpful
- Clear and straightforward
- Optimistic and positive

### Visual Style
- Warm, inviting color palette (yellows, blues, soft pastels)
- Clean, modern design
- Child-friendly but professional
- High-quality imagery
- Plenty of white space

## Design System

### Color Palette
- **Primary**: Golden Yellow (#f9ad00) - represents warmth, sunshine, and optimism
- **Secondary**: Trust Blue (#0080c8) - represents reliability and professionalism
- **Accent Colors**: Soft pink, green, purple, orange - playful, child-friendly
- **Neutrals**: Grays for text and backgrounds

### Typography
- **Headings**: Poppins (bold, friendly, modern)
- **Body**: Inter (clean, readable, professional)
- **Use**: Clear hierarchy, generous line spacing, accessible sizing

### Component Guidelines
- Use rounded corners for friendliness
- Include subtle shadows for depth
- Ensure mobile responsiveness
- Maintain accessibility standards (WCAG 2.1 AA)
- Optimize for fast loading

## Content Guidelines

### Writing Style
- Use simple, clear language
- Avoid jargon
- Focus on benefits, not features
- Include specific examples
- Address parent concerns proactively

### Key Messages
1. Professional websites help daycares grow
2. Local SEO brings families to your door
3. Easy-to-use WordPress platform
4. Affordable solutions for small businesses
5. Specialized expertise in childcare industry

### Content Structure
- Start with the benefit
- Support with evidence or examples
- End with a clear call-to-action
- Use bullet points for scannability
- Include testimonials when possible

## Technical Specifications

### Platform
- WordPress 6.x
- PHP 8.0+
- MySQL 8.0+

### Performance Targets
- Page load time: < 3 seconds
- Mobile PageSpeed score: > 90
- Accessibility score: > 95

### Browser Support
- Chrome (last 2 versions)
- Firefox (last 2 versions)
- Safari (last 2 versions)
- Edge (last 2 versions)

### Device Support
- Mobile: 320px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+

## SEO Guidelines

### Keywords
- Primary: "daycare websites", "childcare website design"
- Local: "[City] daycare", "Valley area childcare"
- Long-tail: "how to create daycare website", "best website features for daycare"

### Meta Descriptions
- 150-160 characters
- Include primary keyword
- Include location
- Include clear benefit

### Content Strategy
- Blog posts: 1-2 per week
- Topics: daycare marketing, web design, local SEO
- Focus on providing value to daycare owners

## Common Use Cases

### For Daycare Owners
1. Learn about website services
2. View portfolio/examples
3. Complete intake form
4. Contact for consultation
5. Access support resources

### For Parents (on client sites)
1. Learn about daycare programs
2. View facility photos
3. Check availability
4. Complete enrollment forms
5. Contact for tours
