# Content Generation Prompts for Valley Daycare Sites

Use these prompts with AI writing assistants to generate consistent, on-brand content for daycare websites.

## Brand Voice Guidelines

Before using any content generation prompt, provide this context:

```
Brand: Valley Daycare Sites
Industry: Web development for childcare providers
Audience: Daycare center owners and administrators
Tone: Friendly, professional, supportive, optimistic
Style: Clear, concise, benefit-focused
Avoid: Corporate jargon, overly technical terms, aggressive sales language
```

## Website Copy Prompts

### Homepage Hero Section
```
Write a compelling hero section for Valley Daycare Sites homepage:

Requirements:
- Main headline (8-12 words)
- Subheadline explaining the value proposition (20-30 words)
- 3-4 feature bullets highlighting key benefits
- Primary CTA: "Get Started" or "Complete Intake Form"
- Secondary CTA: "View Services" or "See Examples"

Focus on:
- How we help daycares connect with local families
- Professional, modern websites
- Easy to use and maintain
- Local SEO benefits

Tone: Warm, welcoming, confident but not boastful
```

## Blog Content Prompts

### Educational Blog Post
```
Write a blog post titled: "[Title]"

Target audience: Daycare center owners
Goal: Provide actionable advice
Length: 800-1200 words

Structure:
1. Introduction (100-150 words)
   - Hook with relatable scenario
   - Present the problem
   - Preview the solution

2. Main Content (600-900 words)
   - 5-7 main points or steps
   - Each with explanation and example
   - Use subheadings for scannability
   - Include relevant statistics or quotes

3. Conclusion (100-150 words)
   - Summarize key takeaways
   - Encourage implementation
   - Include CTA to related service or resource

Style:
- Use "you" to address reader directly
- Include specific examples
- Break up text with bullet points and numbered lists
- Write conversationally but professionally

SEO:
- Include target keyword in title, intro, and 2-3 times in body
- Use H2 and H3 subheadings
- Write compelling meta description (150-160 characters)
```

### Listicle Blog Post
```
Write a listicle: "X [Things/Ways/Tips] to [Achieve Goal]"

Example: "10 Essential Features Every Daycare Website Needs"

For each item:
- Clear title/heading
- 2-3 sentences explanation
- Why it matters
- Quick tip for implementation

Introduction (100 words):
- Set up the list
- Explain why this matters to daycare owners
- What they'll learn

Conclusion (50-75 words):
- Recap value
- CTA: Download checklist, contact us, read related post

Tone: Practical, encouraging, straightforward
Format: Easy to scan, actionable advice
```

### Case Study/Success Story
```
Write a case study for [client daycare name]:

Structure:
1. The Challenge (100-150 words)
   - Client's situation before
   - Specific problems they faced
   - What they needed

2. The Solution (150-200 words)
   - What we created
   - Key features implemented
   - Process overview

3. The Results (100-150 words)
   - Specific, measurable outcomes
   - Client testimonial (quote)
   - Before/after metrics

4. Key Takeaways (50-75 words)
   - Lessons learned
   - How other daycares can benefit

Include:
- Client name and location (if permitted)
- Specific numbers and metrics
- Direct quotes
- Call-to-action: "See how we can help your daycare"

Tone: Factual, celebratory but not exaggerated
```

## SEO Content Prompts

### Landing Page Copy
```
Write SEO-optimized landing page copy for: [specific keyword/service]

Target keyword: [keyword]
Related keywords: [list 3-5 related terms]

Structure:
1. Hero section
   - H1 with target keyword
   - Value proposition
   - CTA above the fold

2. Problem/Solution section (200 words)
   - Address pain points
   - Present solution
   - Include keywords naturally

3. Features/Benefits (300 words)
   - 3-4 main benefits
   - Explain value of each
   - Use semantic keywords

4. Social Proof (100 words)
   - Testimonials or stats
   - Build credibility

5. Final CTA (100 words)
   - Clear action
   - Remove friction
   - Reinforce value

SEO Requirements:
- Keyword density: 1-2%
- Use keyword variations
- Include in title, H1, first paragraph, conclusion
- Write for humans first, search engines second
```

## Usage Guidelines

### Best Practices
1. Always provide specific context about the project
2. Include target audience and goals
3. Specify desired length and format
4. Mention any keywords or phrases to include/avoid
5. Request revisions focusing on specific aspects

### Iteration Process
1. Generate initial draft with AI
2. Review for brand voice and accuracy
3. Edit for specifics and personalization
4. Add client-specific details
5. Final human review and polish

### Quality Checklist
- [ ] Matches brand voice and tone
- [ ] Addresses target audience appropriately
- [ ] Clear and concise
- [ ] Actionable and valuable
- [ ] Free of jargon and errors
- [ ] Includes appropriate CTAs
- [ ] Optimized for intended channel
- [ ] Factually accurate
